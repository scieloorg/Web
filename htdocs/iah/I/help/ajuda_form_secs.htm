the form 		Type a word or more in the fields in the form. If you wish, select the search by consulting a specific index where the words should be looked for.

If you wish to view or select using a specific index, select the field and then click on the index.

To type more than one word in the same line it is necessary to use a boolean operator to relate one to the other (example 1). Use lines 1, 2 and 3 to search more than one word and/or in different fields (example 2).

Operators

To combine the search in more than one word in the same field in the form it is necessary to use boolean operators between the words. The table of boolean operators and examples on how to use them are as follows:
.
  operator

example

function
and cardiology and surgery retrieves all documents which contain the two words: cardiology and surgery
or cardiology or surgery retrieves all documents which contain the word angiologia or the word surgery or documents which have the both words: cardiology and/or surgery or both.
and not cardiology and not surgery retrieves all documents which contain only the word angiologia and do not contain the word surgery: cardiology and not surgery
