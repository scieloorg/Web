patch-SWS-Linux-2007.tgz

A. Este parche tiene como objetivo:

1. Actualizar el procesamiento de envio de bases para BIREME para fines de procesar los enlaces a bases externas

2. Generar y enviar a BIREME cuatro archivos iso: 
	- artigo.iso
	- issues.iso
	- bib4cit.iso
	- title.iso
Para fines de procesar trigrama.

Para verificar cuale archivos serán actualizados, ejecute el comando abajo:

	tar tvfzp patch-SWS-Linux-2007.tgz

B. Para aplicar este parche:

  1. Es deseable que se haga una copia de seguridad del directorio proc o de los archivos que estan en ese parche.

  2. descompacte el archivo con el utilitario tar de acuerdo con el comando:

	tar patch-SWS-Linux-2007.tgz

     en el directorio "proc" (en instalaciones estándares está bajo www).

  3. Abra el archivo /proc/transf/Envia2MedlineLogOn-exemplo.txt y cambie la línea:
	user id password 
     Para 
	user tuUsuario tuContraseña 
     Renombrar el archivo Envia2MedlineLogOn-exemplo.txt para Envia2MedlinePadrao.txt

  4. En el directorio "proc" Ejecutarte el bat Envia2MedlinePadrao.bat
	./Envia2MedlinePadrao.bat
